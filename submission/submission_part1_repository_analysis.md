# Part 1 - Repository Analysis (Task 1.1)

## 1) Identify Python-primary repositories

Using each repo’s GitHub language breakdown (`/languages`), **all five repositories are Python as the largest code language**, but they are **not all “strictly Python-only”** (each contains some non-Python code such as JS/TS, Dockerfile, C/Cython, etc.). Interpreting your requirement as **“Python is the main language”**, **all 5 qualify as Python-primary**.

- aiokafka: Python is dominant (also C/Cython for performance extensions).
- airbyte: Python is the largest, but the repo is clearly polyglot (large Kotlin/Java too).
- archivematica: Python is dominant, with sizeable TS/Vue frontend.
- beets: Python is dominant, some JS assets.
- MetaGPT: Python is dominant, with some TS/JS and shell tooling.
